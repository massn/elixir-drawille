# Elixir-Drawille

Elixir-Drawille is a library for drawings in the terminal using unicode braille characters. It is based on [Drawille](https://github.com/asciimoo/drawille).

## Installation

If [available in Hex](https://hex.pm/docs/publish), the package can be installed as:

  1. Add drawille to your list of dependencies in `mix.exs`:

        def deps do
          [{:drawille, "~> 0.0.1"}]
        end

  2. Ensure drawille is started before your application:

        def application do
          [applications: [:drawille]]
        end
